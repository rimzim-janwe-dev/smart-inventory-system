# Smart Inventory System

## 🚀 Project Overview
Smart Inventory Management System with Restock Alerts (AI-like feature)

## 🛠 Tech Stack
- ASP.NET Core Web API
- Entity Framework Core
- MySQL
- C#

## ✨ Features
- Product CRUD Operations
- Low Stock Detection (Restock Alerts)
- Clean Architecture (Controller-Service Pattern)
